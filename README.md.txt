Commercial Rent Prediction

This project predicts rent prices for commercial properties using features like 
city, area, type of commercial property, and size. The entire model development was done in a Jupyter Notebook, 
and the app is deployed via Streamlit on Render.

 
Live App
-> [https://commercial-streamlit.onrender.com]

Project Overview
- Commercial real estate dataset
- Data cleaning and preprocessing pipeline
- Regression modeling (Random Forest or other used model)
- Streamlit frontend for live prediction

Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Fast API
- Jupyter Notebook(anaconda)
